# 快速开始

## 1. 确认使用场景

先确定需要连接的是 AI IDE、Agent 工作流、MCP 客户端还是 LightRAG。

## 2. 打开项目

<ProjectLink href="https://graph.tohoqing.com/">进入 FluctGraph</ProjectLink>

## 3. 使用项目当前提供的接入信息

从项目页面获取当前版本的安装、地址与配置说明。不要根据旧截图或第三方示例填写密钥和服务地址。

## 4. 从最小知识库验证

先使用少量非敏感内容完成连接和检索验证，再逐步扩展私有知识库范围。
