# 快速开始

## 1. 准备 API Key

在页面的 Access 区域填写可用的 API Key。不要在共享设备上长期保留密钥。

## 2. 选择模型与引擎

从页面当前提供的模型与引擎中选择适合的选项。

## 3. 输入提示词

清楚描述主体、场景、风格、构图与必须避免的内容。

## 4. 生成并检查结果

生成第一张图片后，检查主体、构图、文字、边缘和不希望出现的元素。

## 5. 按需编辑

使用参考图、遮罩或编辑功能进行一次有针对性的修改，避免同时改变过多条件。
