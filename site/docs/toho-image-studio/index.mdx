# Toho Image Studio

Toho Image Studio 是面向图像生成与编辑工作流的浏览器创作空间。

公开界面提供 API Key、模型与引擎选择、提示词输入、参考图、遮罩、生成结果、编辑、历史结果和直连接口等区域。
